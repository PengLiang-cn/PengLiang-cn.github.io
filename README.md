# Peng Liang's Personal Website

A modern, responsive personal academic website for Peng Liang, Ph.D. candidate at University of Electronic Science and Technology of China.

## Features

### 🎨 **Visual Enhancements**
- Modern, clean design with improved typography
- Responsive layout for all device sizes
- Smooth animations and transitions
- Interactive hover effects
- Dark theme support (optional)

### 📊 **Content Sections**
1. **About Me** - Personal introduction and background
2. **Publications** - Academic papers with links
3. **Competition** - Awards and achievements
4. **Projects** - Research and development projects
5. **Skills** - Technical expertise and tools
6. **Contact** - Contact information and social links

### ⚡ **Performance Optimizations**
- Custom CSS for faster loading
- JavaScript enhancements for better UX
- Smooth scrolling navigation
- Lazy loading images
- Optimized animations

### 🛠 **Technical Stack**
- HTML5, CSS3, JavaScript
- Academic theme (Hugo-based)
- Bootstrap 4 for responsive design
- Font Awesome icons
- Google Analytics integration

## Setup & Customization

### Local Development
1. Clone the repository
2. Open `index.html` in a web browser
3. For full Hugo development, install Hugo and run `hugo server`

### Content Updates
- **Profile info**: Edit `index.html` lines 600-700
- **Publications**: Edit `index.html` lines 730-1260
- **Skills**: Edit `index.html` Skills section
- **Projects**: Edit `index.html` Projects section
- **Styling**: Edit `css/custom.css`
- **Interactivity**: Edit `js/enhancements.js`

### Adding New Features
1. **New section**: Add HTML structure in `index.html`
2. **Styling**: Add CSS rules in `css/custom.css`
3. **Interactivity**: Add JavaScript in `js/enhancements.js`
4. **Images**: Place in appropriate folder and update paths
